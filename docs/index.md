# AI Tamer Documentation

Welcome to the AI Tamer project documentation. This plugin aims to solve the "All or Nothing" dilemma where website owners must choose between blocking AI bots (losing SEO traffic) or allowing them (permitting unauthorized training).

## Table of Contents

1. [Architecture Overview](architecture.md) - How AI Tamer approaches the problem.
2. [Technical Specifications](technical-spec.md) - Details on implementation strategy.
3. [Project Roadmap](roadmap.md) - MVP stages and future goals.

## Core Problem
The modern web faces a crisis where generative AI models consume content without compensation, yet blocking them can destroy search engine visibility. AI Tamer provides a "triage" system to manage this relationship programmatically.
